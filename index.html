Mistral sort Devstral 2

Production de code Sécurité applicative

Mistral AI publie deux modèles open source dédiés au développement logiciel : Devstral2 (123B) et Devstral Small2 (24B), capables de rivaliser avec des modèles beaucoup plus grands. Devstral2 peut analyser et modifier des bases de code entières, suivre les dépendances et corriger automatiquement des erreurs. Devstral Small2 fonctionne localement avec des boucles rapides et peut alimenter des agents multimodaux. Les deux modèles sont optimisés pour l'écosystème Nvidia sous licences permissives. Mistral lance également Vibe CLI, un assistant en ligne de commande capable d'explorer, modifier et exécuter du code en langage naturel, d'interagir avec Git et de réduire le temps des pull requests.

OpenAI et AWS : partenariat stratégique

Production de code Tests & qualité logicielle

OpenAI et AWS concluent un accord massif dans lequel Amazon investit 50 milliards de dollars et obtient l'exclusivité de la distribution cloud tierce pour Frontier, la plateforme dédiée aux AI coworkers. OpenAI s'engage à utiliser largement l'infrastructure AWS, notamment 2GW de capacité Trainium, afin de soutenir le développement du futur Stateful Runtime Environment que les deux entreprises vont co-concevoir. Ce partenariat inclut la création de modèles personnalisés destinés aux services d'Amazon. L'objectif est de disposer d'un environnement d'exécution capable de faire tourner des applications et agents d'IA générative à grande échelle.
