Présentation

Qu'est-ce que CyberNews ?

CyberNews est un site web dynamique développé en PHP et MySQL dans le cadre du BTS SIO. Son objectif : centraliser et présenter des articles de veille sur la cybersécurité, organisés par catégories (fuites de données, cyberattaques, dark web, CNIL…).

Le site se connecte à une base de données relationnelle pour afficher dynamiquement les news et leurs détails, avec pour chaque article un résumé, un lien source et un PDF téléchargeable.

Base de données

Schéma relationnel

La base eko_cybernews contient deux tables liées par une clé étrangère.

TABLE — categorie
IDCategorie INT · PK · AUTO_INCREMENT
Categorie VARCHAR(38)
TABLE — news
IDNews INT · PK · AUTO_INCREMENT
IDCategorie INT · FK
Titre VARCHAR(80)
Auteur VARCHAR(38)
Date DATE
URL VARCHAR(300)
Resume TEXT
PDF VARCHAR(300)

news.IDCategorie → FK → categorie.IDCategorie

Extrait de code

Connexion à la base de données

Chaque page se connecte à MySQL via mysqli_connect(), puis ferme la connexion après usage.

EKO_index.php 
// Connexion à la base de données eko_cybernews
$EKO_connexion = mysqli_connect(
    "localhost",  // hôte
    "root",        // utilisateur
    "",            // mot de passe (local)
    "eko_cybernews" // nom de la BDD
);

// Inclusion du menu dynamique (requiert la connexion active)
include "include/EKO_menu.inc.php";

// Fermeture propre de la connexion
$EKO_connexion->close();

Menu dynamique par catégories

Le fichier EKO_menu.inc.php génère automatiquement le menu en parcourant les catégories et les news de la BDD avec deux boucles imbriquées.

include/EKO_menu.inc.php 
// 1. Récupère toutes les catégories triées A→Z
$EKO_reqCat = "SELECT IDCategorie, Categorie
               FROM categorie
               ORDER BY Categorie ASC";
$EKO_toutesCat = $EKO_connexion->query($EKO_reqCat);

// 2. Pour chaque catégorie : affiche son titre + ses news
while ($EKO_uneCat = $EKO_toutesCat->fetch_assoc()) {
    echo "<h3>".$EKO_uneCat["Categorie"]."</h3><ul>";

    $EKO_idCat  = $EKO_uneCat["IDCategorie"];
    $EKO_reqNews = "SELECT IDNews, Titre FROM news
                    WHERE IDCategorie = $EKO_idCat
                    ORDER BY Date DESC";
    $EKO_toutesNews = $EKO_connexion->query($EKO_reqNews);

    // 3. Génère un lien par article
    while ($EKO_uneNews = $EKO_toutesNews->fetch_assoc()) {
        echo "<li><a href='EKO_news.php?id="
             .$EKO_uneNews["IDNews"]
             ."'>→ ".$EKO_uneNews["Titre"]."</a></li>";
    }
    echo "</ul>";
}

Affichage d'un article (INNER JOIN)

La page EKO_news.php reçoit l'ID de l'article via $_GET et exécute un INNER JOIN pour récupérer à la fois les données de la news et le nom de sa catégorie.

EKO_news.php 
// Récupération de l'ID passé en paramètre GET
$EKO_id = $_GET["id"];

// Requête avec INNER JOIN pour lier news et catégorie
$EKO_req = "SELECT n.IDNews, n.Titre, n.Auteur,
                   n.Date,   n.URL,    n.Resume,
                   n.PDF,    c.Categorie
            FROM   news      n
            INNER JOIN categorie c
                   ON n.IDCategorie = c.IDCategorie
            WHERE  n.IDNews = $EKO_id";

$EKO_Lignenews = $EKO_connexion
    ->query($EKO_req)
    ->fetch_assoc();

// Affichage des données récupérées
echo $EKO_Lignenews["Titre"];
echo $EKO_Lignenews["Categorie"];
echo $EKO_Lignenews["Resume"];
